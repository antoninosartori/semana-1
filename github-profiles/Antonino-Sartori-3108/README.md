## Comandos de la terminal

| Comando | Descripcion |
| --- | --- |
| `mkdir` | Crea una nueva carpeta |
| `cd` | Cambia el directorio |
| `ls` | Muestra los archivos de ese directorio |
| `rm` | Elimina un archivo determinado |
| `cp` | Copia un archivo determinado |

| Alias | Comando | Descripcion |
| --- | --- | --- |
| cl | `clear` | Limpia la terminal
| .. | `cd..` | Vuelve al directorio padre
